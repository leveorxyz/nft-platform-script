# NFT platform script

Scripts to compile, deploy and test NFT contracts in multiple EVM based blockchain on testnet and mainnet.

## Used technology
- Typescript
- Typechain
- Solidity
- Hardhat

## Instruction
- Compile:
```
npm run compile
```
- Run test:
```
npm run test
```
- Deploy
```
npm run deploy:<network>
```
- Verify on etherscan
```
npx hardhat verify --network rinkeby <YOUR_CONTRACT_ADDRESS>
```
